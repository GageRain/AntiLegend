## SpringBoot ＆ VUE 资料共享

SpringBoot 是由Pivotal团队提供的全新框架，其设计目的是用来简化Spring应用搭建和开发过程的一种框架

Vue 是一套用户构建用户界面的渐进式框架，是自底向上逐层应用

SpringBoot + Vue 的前后端分离式开发，是Web信息系统最流行的开发模式之一
